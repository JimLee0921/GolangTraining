## WriteString 写字符串

WriteString 写入一个字符串。返回写入的字节数。如果写入字节数小于 len(s)，还会返回一个错误，解释写入长度不足的原因。

### 方法签名

```
func (b *Writer) WriteString(s string) (int, error)
```

参数：

- s: 要写入的字符串内容

返回值：

- n：实际写入缓冲区的字节数，不是字符数（UTF-8 多字节字符会更大），若部分写入则 `n < len(s)`
- err：错误（若底层写入失败）

### 核心功能

将一个字符串 s 写入到缓冲区中（而不是直接写到底层），返回写入的字节数和可能的错误。

核心区别于 `Write()`：

- `Write()` 接收 字节切片（`[]byte`）
- `WriteString()` 接收 字符串
- 其它行为完全一致（同样需要 `Flush()` 才真正写出）

### 性能差异

当调用

```
w.Write([]byte("hello"))
```

Go 会先把字符串 "hello" 转换成新的 []byte 临时切片，这会：

- 多一次内存分配
- 多一次数据复制

而

```
w.WriteString("hello")
```

在内部是直接基于字符串底层的字节数据写入缓冲区，不会额外创建临时切片。

> `WriteString()` 更快；更节省内存；尤其适合频繁写文本的场景（日志、文件输出等）。
