## WriteByte 写入字节

常用于手动构造输出流（例如文件格式、协议数据、CSV 构建器）时

### 方法签名

```
func (b *Writer) WriteByte(c byte) error
```

参数：

- c：要写入的单个字节（即 uint8）

返回值

- error：若写入失败则返回错误（底层错误或 Flush 失败）

### 功能说明

`WriteByte()` 会将单个字节写入缓冲区（不是直接写到文件）。
相当于：`Write()` 的精简版，专门用于写入单个字节。
当缓冲区满时，会自动调用 `Flush()`，把缓冲区内容写出到底层。

### 注意事项

1. 仍需手动 `Flush()`
    - 即使写了很多单字节，也必须最后调用 `Flush()`
    - 否则最后那部分缓冲内容不会写出

2. 不能写中文（多字节）
    - byte 只能表达单字节字符（ASCII）
    - 如果要写 "中" 这样的多字节 UTF-8 字符需要用 `WriteRune()`

3. 需要错误检查
    - 如果底层写入失败（例如磁盘错误），WriteByte 会返回错误
