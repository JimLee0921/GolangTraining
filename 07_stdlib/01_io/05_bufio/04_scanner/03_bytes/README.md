## Bytes

`Scanner.Bytes()` 是 `Scanner.Text()` 的底层兄弟方法，功能几乎一样，只是返回类型不同，使用场景略有差别。
主要用于处理二进制数据或性能优化时。

### 方法签名

```
func (s *Scanner) Bytes() []byte
```

### 功能说明

`Bytes()` 返回最近一次成功调用 `Scan()` 时，扫描到的数据片段的 字节切片（`[]byte`）。

- `Text()` -> 返回 字符串副本
- `Bytes()` -> 返回 底层原始字节切片

### 对比

| 对比项       | `Text()`    | `Bytes()`            |
|-----------|-------------|----------------------|
| 返回类型      | `string`    | `[]byte`             |
| 是否拷贝数据    | 会拷贝（产生新字符串） | 不拷贝（直接指向内部缓冲）        |
| 是否可修改     | 不可修改        | 可以修改（不推荐）            |
| 是否适合二进制数据 | 否           | 是                    |
| 是否安全长期保存  | 可以          | 不可以（下次 `Scan()` 就失效） |
