## ReadSlice 读取切片

### 方法定义

```
func (b * Reader ) ReadSlice(delim byte ) (line [] byte，err error )
```

参数：

- `delim byte`：要查找的分隔符（单个字节）
- 例如：'\n' 表示读到换行为止，',' 表示逗号分隔

返回值：

- line：`[]byte`类型，返回的切片（可能共享内部缓冲区）
- err： 错误：可能是
    - `nil`：成功找到分隔符
    - `io.EOF`：到达输入末尾（可能返回部分数据）
    - `ErrBufferFull`：缓冲区满但仍未找到分隔符，需要自行处理

### 核心功能

从缓冲区和底层读取数据，直到遇到指定分隔符 delim，返回包括分隔符在内的数据。

- 如果 delim = '\n'，就会返回一行字符串
- 如果 delim = ','，就会返回一个逗号分隔的字段

### 注意事项

ReadSlice() 返回的 []byte 直接指向内部缓冲区，不会拷贝。
返回切片共享缓冲区意味着：

- 下次任何读取操作都会覆盖这个切片的内容
- 如果想长期保存结果，必须手动拷贝一份

```
copyBuf := append([]byte(nil), line...)
```

另外如果处理过长的行可能会导致缓冲区溢出，建议在创建 reader 时手动指定缓冲区大小
