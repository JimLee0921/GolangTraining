# 路径处理模块 path

在 Go 中，路径操作主要分为两组包

| 包名              | 适用平台                 | 说明                                          |
|-----------------|----------------------|---------------------------------------------|
| `path`          | 跨平台逻辑（主要用于 URL、虚拟路径） | 操作 **“/” 分隔的路径**                            |
| `path/filepath` | **系统文件路径**           | 会自动使用系统分隔符（Windows 用 `\`，Linux/macOS 用 `/`） |

> path 官方文档：https://pkg.go.dev/path
>
> filepath 官方文档：https://pkg.go.dev/path/filepath

## 差异对比

两个包有些重复方法

| 函数                     | `path` 特性      | `filepath` 特性 |
|------------------------|----------------|---------------|
| `Base` / `Dir` / `Ext` | 简单字符串处理，固定 `/` | 平台感知，智能识别 `\` |
| `Join` / `Clean`       | 逻辑路径拼接（Web）    | 文件系统安全拼接      |
| `IsAbs`                | 仅判断 `/` 开头     | 判断绝对路径（含盘符）   |
| `Match`                | 固定 `/`         | 兼容系统分隔符       |

## 选择时机

| 目标                 | 推荐包                | 理由              |
|--------------------|--------------------|-----------------|
| Web / API / URL 路径 | `path`             | 与操作系统无关，永远用 `/` |
| 文件系统路径             | `filepath`         | 自动适配分隔符，兼容跨平台   |
| 混合环境（如：项目构建脚本）     | **优先用 `filepath`** | 保证不同系统都能正确运行    |
