## 高阶函数

高阶函数（Higher-Order Function） 是 Go 中理解函数式编程思想的关键概念之一，高阶函数就是参数或返回值中包含函数的函数

如果一个函数满足以下任意一种情况，就叫高阶函数：

- 接收函数作为参数
- 返回一个函数作为结果

这两种情况都属于函数操作函数，即函数是一等公民（first-class citizen）

## 函数工厂

函数工厂（Function Factory）是高阶函数的一种典型应用形式，
是理解闭包与函数返回函数的基础

函数工厂（Function Factory）：是一种 返回函数的函数（高阶函数），
用于根据输入参数动态生成具有特定行为的函数

> 写一个函数，这个函数不直接执行逻辑，而是制造出另一个带有特定配置的函数

### 思想

- 工厂函数（factory function） -> 生产新的函数
- 返回的函数通常是一个闭包（closure）
- 闭包能捕获外部变量，从而带着参数状态工作
- 每次调用工厂函数，都会生成一个独立的函数实例
