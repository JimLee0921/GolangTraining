# 数值类型

Go 对数值类型主要分为 整数类型、浮点类型 和 复数类型

## 整数类型（Integers）

整数类型又分为 **有符号整数** 和 **无符号整数**：

| 类型      | 位数                | 取值范围                                                   |
|---------|-------------------|--------------------------------------------------------|
| `int8`  | 8位                | -128 ～ 127                                             |
| `int16` | 16位               | -32768 ～ 32767                                         |
| `int32` | 32位               | -2,147,483,648 ～ 2,147,483,647                         |
| `int64` | 64位               | -9,223,372,036,854,775,808 ～ 9,223,372,036,854,775,807 |
| `int`   | 与平台有关，32位或64位（常用） |                                                        |

无符号整数（只能是非负数）：

| 类型       | 位数            | 取值范围                           |
|----------|---------------|--------------------------------|
| `uint8`  | 8位            | 0 ～ 255                        |
| `uint16` | 16位           | 0 ～ 65535                      |
| `uint32` | 32位           | 0 ～ 4,294,967,295              |
| `uint64` | 64位           | 0 ～ 18,446,744,073,709,551,615 |
| `uint`   | 与平台有关，32位或64位 |                                |

特殊整数：

* `byte` 是 `uint8` 的别名（常用于存储单个字符）
* `rune` 是 `int32` 的别名（常用于存储 Unicode 字符）

### 示例

```go
package main

import "fmt"

func main() {
	var a int = 100
	var b uint = 200
	var c byte = 'A'
	var d rune = '中'

	fmt.Println(a, b, c, d) // 100 200 65 20013
}
```

## 浮点类型（Floats）

浮点数表示带小数的数值：

| 类型        | 精度  | 说明                     |
|-----------|-----|------------------------|
| `float32` | 32位 | 大约 6\~7 位十进制有效数字       |
| `float64` | 64位 | 大约 15\~16 位十进制有效数字（推荐） |

默认情况下，小数会被推断为 `float64`。

### 示例

```go
package main

import "fmt"

func main() {
	var f1 float32 = 3.14
	var f2 float64 = 2.718281828459045

	fmt.Println(f1) // 3.14
	fmt.Println(f2) // 2.718281828459045
}
```

## 复数类型（Complex Numbers）

Go 还原生支持复数类型：

| 类型           | 说明                 |
|--------------|--------------------|
| `complex64`  | 实部和虚部分别是 `float32` |
| `complex128` | 实部和虚部分别是 `float64` |

可以用 `complex(re, im)` 来创建，或者用 `a + bi` 的形式。

### 示例

```go
package main

import "fmt"

func main() {
	c1 := complex(2, 3) // complex128 (2+3i)
	c2 := 1 + 2i        // complex128

	fmt.Println(c1)       // (2+3i)
	fmt.Println(c2)       // (1+2i)
	fmt.Println(real(c1)) // 2 (取实部)
	fmt.Println(imag(c1)) // 3 (取虚部)
}
```

## 数值运算符

常见运算符适用于整数和浮点数：

* `+` 加法
* `-` 减法
* `*` 乘法
* `/` 除法
* `%` 取余（仅整数）

示例：

```go
x := 10
y := 3
fmt.Println(x / y) // 3
fmt.Println(x % y) // 1
```

---

## 类型转换

Go **不会自动类型提升**，不同类型必须显式转换：

```go
var i int32 = 100
var j int64 = int64(i) // 必须显式转换
```
