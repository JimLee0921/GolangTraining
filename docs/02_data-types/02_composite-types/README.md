# 复合类型总览

> 复合类型通过组合多个值描述更复杂的数据结构，是 Go 程序建模的核心。

## 复合类型分类

- **数组**：长度固定，理解值语义与多维数组声明方式。
- **切片**：基于数组实现，掌握容量、扩容及共享底层数组的影响。
- **映射**：适合键值对存储，注意并发安全与存在性检查。
- **结构体**：表达对象模型，关注字段标签、匿名字段以及比较规则。
- **指针**：直接操作内存地址，用于修改外部变量或避免大对象拷贝。
- **函数**：一等公民，支持多返回值、闭包、可变参数与回调。
- **通道**：并发通信工具，结合 goroutine 实现安全的数据传递。
- **接口**：定义行为抽象，支持多态与依赖倒置，是 Go 设计的核心。
