# 切片

> 具体代码见：[slice切片](../../../02_data-type/03_composite-types/02_slice)

## 核心概念

- 切片是对底层数组的动态视图，由指针、长度、容量组成。
- 支持自动扩容，常用作 Go 中的动态数组。

## 语法要点

- 切片表达式 `a[low:high:max]` 控制长度与容量。
- 使用 `make([]T, len, cap)` 创建具备初始容量的切片。

## 实用示例

- 实现栈/队列、批量处理数据。
- 利用 `copy` 函数进行内存移动。

## 常见陷阱

- 共享底层数组易导致意外修改；必要时显式复制。
- 切片增长策略可能触发重新分配，注意性能分析。
