# Hello Go

## fmt 输出相关

### Go `fmt` 打印函数

一共有三种 Print 函数 `Print`、`Sprint` 和 `Fprint`

| 家族         | 函数         | 作用                        | 示例结果（假设 name := "Gopher", n := 42）                        |
|------------|------------|---------------------------|-----------------------------------------------------------|
| **Print**  | `Print`    | 直接打印，不换行                  | `fmt.Print(name, n)` → 控制台输出 `Gopher42`                   |
|            | `Println`  | 打印，参数间加空格，末尾换行            | `fmt.Println(name, n)` → 控制台输出 `Gopher 42\n`              |
|            | `Printf`   | **格式化**打印到控制台             | `fmt.Printf("%s-%d\n", name, n)` → 控制台输出 `Gopher-42\n`    |
| **Sprint** | `Sprint`   | 返回字符串（拼接），无换行             | `fmt.Sprint(name, n)` → `"Gopher42"`                      |
|            | `Sprintln` | 返回字符串（拼接），末尾带换行           | `fmt.Sprintln(name, n)` → `"Gopher 42\n"`                 |
|            | `Sprintf`  | **格式化**后返回字符串             | `fmt.Sprintf("%s-%d", name, n)` → `"Gopher-42"`           |
| **Fprint** | `Fprint`   | 写入到 `io.Writer`（如文件/网络连接） | `fmt.Fprint(file, name, n)` → 文件中写入 `Gopher42`            |
|            | `Fprintln` | 写入到 `io.Writer`，带空格和换行    | `fmt.Fprintln(file, name, n)` → 文件中写入 `Gopher 42\n`       |
|            | `Fprintf`  | **格式化**后写入 `io.Writer`    | `fmt.Fprintf(file, "%s-%d", name, n)` → 文件中写入 `Gopher-42` |

### Go 常用格式化动词

下面为 fmt `printf`
函数常用格式化动词。使用案例见 [format-verbs/main.go](../01_getting-started/03_print-functions/format-verbs/main.go)。

| 类别      | 动词     | 说明                         | 示例输出                   |
|---------|--------|----------------------------|------------------------|
| **整数**  | `%d`   | 十进制                        | `42`                   |
|         | `%b`   | 二进制                        | `101010`               |
|         | `%o`   | 八进制                        | `52`                   |
|         | `%x`   | 十六进制（小写）                   | `2a`                   |
|         | `%X`   | 十六进制（大写）                   | `2A`                   |
| **浮点**  | `%f`   | 小数形式，默认 6 位小数              | `3.141593`             |
|         | `%.2f` | 小数形式，保留 2 位                | `3.14`                 |
|         | `%e`   | 科学计数法（小写 e）                | `3.141593e+00`         |
|         | `%E`   | 科学计数法（大写 E）                | `3.141593E+00`         |
|         | `%g`   | 自动选择 `%f` 或 `%e`（更紧凑，小写 e） | `3.14159` / `1.23e+08` |
|         | `%G`   | 自动选择 `%f` 或 `%E`（更紧凑，大写 E） | `3.14159` / `1.23E+08` |
| **字符串** | `%s`   | 原样输出字符串                    | `hello`                |
|         | `%q`   | 带双引号并转义                    | `"hello\nworld"`       |
|         | `%x`   | 字符串/字节切片转十六进制（小写）          | `68656c6c6f` (`Hello`) |
| **布尔**  | `%t`   | 布尔值                        | `true` / `false`       |
| **通用**  | `%v`   | 默认格式                       | `{1 2}`                |
|         | `%+v`  | 结构体带字段名                    | `{X:1 Y:2}`            |
|         | `%#v`  | Go 源码表示                    | `main.Point{X:1, Y:2}` |
|         | `%T`   | 类型                         | `main.Point`           |
