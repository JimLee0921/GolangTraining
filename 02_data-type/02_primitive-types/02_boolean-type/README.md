# 布尔类型（Boolean Type）

## 概述

布尔类型（`bool`）是 Go 中用于表示逻辑真值的数据类型
它只有两个取值：

* `true` —— 表示“真”
* `false` —— 表示“假”

布尔类型是控制语句（如 `if`、`for`、`switch`）的基础，用于表达条件与逻辑判断

---

## 类型特征

* Go 中的布尔类型只有一个：`bool`
* 零值为 `false`
* 不支持使用 `0` / `1` 代替布尔值
* 不支持算术运算（例如不能执行 `true + false`）

---

## 布尔表达式来源

布尔值通常来源于以下几类表达式：

1. **比较运算**

    * `==`, `!=`, `<`, `<=`, `>`, `>=`
    * 返回结果为 `bool`

2. **逻辑运算**

    * 通过逻辑运算符组合多个条件（见下）

3. **显式声明或赋值**

    * 直接写 `true` / `false`
    * 从函数返回结果中获取

---

## 逻辑运算符

| 运算符    | 名称       | 说明                          |   
|--------|----------|-----------------------------|
| `&&`   | 逻辑与（AND） | 两个表达式都为 `true` 时，结果为 `true` |   
| `\|\|` | 逻辑或（OR）  | 至少一个为 `true`，结果为 `true`     |   
| `!`    | 逻辑非（NOT） | 将 `true` 取反为 `false`，反之亦然   |   

**短路求值（Short-circuit Evaluation）**

* `a && b`：若 `a` 为 `false`，不会计算 `b`
* `a || b`：若 `a` 为 `true`，不会计算 `b`

这种特性常用于避免多余计算或错误调用（例如空指针检查）

---

## 布尔类型在控制结构中的使用

1. **条件判断**

    * `if` 后的表达式必须是布尔值；
    * 不允许使用非布尔类型（如数字或字符串）。

2. **循环控制**

    * `for` 语句可以直接基于布尔条件；
    * 当条件为 `false` 时，循环终止。

3. **逻辑组合**

    * 可以嵌套或组合多个布尔条件；
    * 常见于多条件判断（如权限验证、范围检测）。

---

## 语言特性与约束

| 特性      | 描述                           |
|---------|------------------------------|
| 类型安全    | 不允许自动转换为其他类型（如 `int → bool`） |
| 无隐式类型提升 | 表达式必须严格返回 `bool`             |
| 不支持位运算  | 位逻辑仅适用于整数类型                  |
| 支持比较    | `==` 与 `!=` 可用于布尔值比较         |

---

## 零值与初始化

* 零值为 `false`
* 声明但未赋值时自动为 `false`
* 在逻辑判断中，零值相当于条件不成立
