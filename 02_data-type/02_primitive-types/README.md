# Go 基础数据类型

## 概述

Go 的基础类型包括：

* 数值类型（整数、浮点数、复数）
* 布尔类型（bool）
* 字符与字符串类型（byte、rune、string）

## 总览

| 类型类别 | 示例             | 关键特性                |
|------|----------------|---------------------|
| 整数   | `int`, `int64` | 离散数值、无小数            |
| 浮点   | `float64`      | 支持小数，可能有精度误差        |
| 复数   | `complex128`   | 包含实部与虚部             |
| 布尔   | `bool`         | 真/假逻辑判断             |
| 字符   | `byte`, `rune` | 单个字符（ASCII/Unicode） |
| 字符串  | `string`       | 不可变字节序列             |
