Go 语言中的运算符（Operators）分为 **六大类**，共 40 多个符号

---

## 算术运算符（Arithmetic Operators）

| 运算符 | 含义           | 示例       | 结果   |
|:---:|:-------------|:---------|:-----|
| `+` | 加法           | `10 + 3` | `13` |
| `-` | 减法           | `10 - 3` | `7`  |
| `*` | 乘法           | `10 * 3` | `30` |
| `/` | 除法（整数除法向下取整） | `10 / 3` | `3`  |
| `%` | 取余（mod）      | `10 % 3` | `1`  |

注意：

* `/` 对整数类型会执行整除；若要浮点除法需显式使用浮点数：`10.0 / 3.0`
* `%` 仅支持整数类型

---

## 关系（比较）运算符（Relational Operators）

| 运算符  | 含义   | 示例       | 结果      |
|:----:|:-----|:---------|:--------|
| `==` | 等于   | `3 == 3` | `true`  |
| `!=` | 不等于  | `3 != 2` | `true`  |
| `>`  | 大于   | `5 > 2`  | `true`  |
| `<`  | 小于   | `5 < 2`  | `false` |
| `>=` | 大于等于 | `5 >= 5` | `true`  |
| `<=` | 小于等于 | `5 <= 4` | `false` |

类型必须相同（Go 不自动类型转换），例如：

```go
var a int = 5
var b float64 = 5.0
fmt.Println(a == b) // 编译错误
```

## 逻辑运算符（Logical Operators）

| 运算符  | 含义       | 示例              | 结果      |       |   |        |        |
|:----:|:---------|:----------------|:--------|-------|---|--------|--------|
| `&&` | 逻辑与（AND） | `true && false` | `false` |       |   |        |        |
|  `   |          | `               | 逻辑或（OR） | `true |   | false` | `true` |
| `!`  | 逻辑非（NOT） | `!true`         | `false` |       |   |        |        |

特点：

* `&&` 和 `||` **短路求值（short-circuit evaluation）**

    * 若 `A && B` 中 `A` 为 `false`，不会计算 `B`
    * 若 `A || B` 中 `A` 为 `true`，不会计算 `B`

## 位运算符（Bitwise Operators）

| 运算符  | 含义           | 示例       | 二进制运算          | 结果  |      |      |     |
|:----:|:-------------|:---------|:---------------|:----|------|------|-----|
| `&`  | 按位与          | `6 & 3`  | `110 & 011`    | `2` |      |      |     |
|  `   | `            | 按位或      | `6             | 3`  | `110 | 011` | `7` |
| `^`  | 按位异或         | `6 ^ 3`  | `110 ^ 011`    | `5` |      |      |     |
| `&^` | 位清除（AND NOT） | `6 &^ 3` | `110 &^ 011`   | `4` |      |      |     |
| `<<` | 左移           | `3 << 1` | `0110`         | `6` |      |      |     |
| `>>` | 右移           | `8 >> 2` | `1000 -> 0010` | `2` |      |      |     |

特殊说明：

* `^` 在不同语境中有两种含义：

    * **按位异或**（二元）-> `a ^ b`
    * **按位取反**（一元）-> `^a`
* `&^`（AND NOT）表示把右操作数对应位为 1 的位清除
  示例：
  ```go
  fmt.Println(6 &^ 3) // 4
  // 110 &^ 011 = 100 (4)
  ```

## 赋值运算符（Assignment Operators）

|  运算符  | 含义      | 示例        | 等价写法         |      |        |    |
|:-----:|:--------|:----------|:-------------|------|--------|----|
|  `=`  | 赋值      | `a = 5`   | —            |      |        |    |
| `+=`  | 加并赋值    | `a += 3`  | `a = a + 3`  |      |        |    |
| `-=`  | 减并赋值    | `a -= 2`  | `a = a - 2`  |      |        |    |
| `*=`  | 乘并赋值    | `a *= 4`  | `a = a * 4`  |      |        |    |
| `/=`  | 除并赋值    | `a /= 2`  | `a = a / 2`  |      |        |    |
| `%=`  | 取余并赋值   | `a %= 3`  | `a = a % 3`  |      |        |    |
| `<<=` | 左移并赋值   | `a <<= 1` | `a = a << 1` |      |        |    |
| `>>=` | 右移并赋值   | `a >>= 1` | `a = a >> 1` |      |        |    |
| `&=`  | 按位与并赋值  | `a &= 2`  | `a = a & 2`  |      |        |    |
|   `   | =`      | 按位或并赋值    | `a           | = 3` | `a = a | 3` |
| `^=`  | 按位异或并赋值 | `a ^= 3`  | `a = a ^ 3`  |      |        |    |

## 其他运算符（Miscellaneous）

| 运算符  | 含义            | 示例                 | 说明         |
|:----:|:--------------|:-------------------|:-----------|
| `&`  | 取地址           | `p := &a`          | 获取变量地址     |
| `*`  | 指针解引用         | `b := *p`          | 访问指针指向的值   |
| `<-` | 通道（Channel）操作 | `<-ch` / `ch <- v` | 读 / 写通道    |
| `,`  | 多重赋值分隔        | `a, b = 1, 2`      | 支持交换变量     |
| `()` | 调用或分组         | `f()` / `(a+b)*c`  | 调函数或表达式优先级 |
| `[]` | 索引            | `arr[2]`           | 取数组/切片元素   |
| `{}` | 结构体/字面量定义     | `p := Person{}`    | 初始化        |
| `.`  | 成员访问          | `obj.Field`        | 访问结构体字段或方法 |

## 运算符优先级（从高到低）

| 优先级 | 运算符                            |       |   |
|:---:|:-------------------------------|-------|---|
|  1  | `()`、`[]`、`.`                  |       |   |
|  2  | `++`、`--`（仅表达式，不是语句）           |       |   |
|  3  | `*`、`/`、`%`、`<<`、`>>`、`&`、`&^` |       |   |
|  4  | `+`、`-`、`                      | `、`^` |   |
|  5  | 比较：`==`、`!=`、`<`、`<=`、`>`、`>=` |       |   |
|  6  | 逻辑与：`&&`                       |       |   |
|  7  | 逻辑或：`                          |       | ` |
|  8  | 赋值类：`=`、`+=`、`-=` 等            |       |   |

## 常见陷阱与注意点

1. Go 中没有 `++` 和 `--` 的前缀形式（只能后缀）

   ```go
   i++  
   ++i  // 编译错误
   ```

2. 逻辑运算符仅能用于布尔类型，不能用于整数

3. `/` 对整数会自动取整，不会四舍五入

   ```go
   fmt.Println(5 / 2) // 2
   ```

4. 通道操作 `<-` 也是运算符，而非语法糖
